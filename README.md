# Wojciech Droździk — strona (Part Time PM + automatyzacje procesów)

Statyczna strona wizytówka. Pojedynczy plik `index.html` (self-contained bundle — czcionki i assety osadzone w pliku).

## Hosting

Hostowana na **Cloudflare Pages**. Każdy push do gałęzi `main` automatycznie publikuje nową wersję.

## Podgląd lokalny

```bash
# dowolny statyczny serwer, np.:
python3 -m http.server 8080
# potem otwórz http://localhost:8080
```
